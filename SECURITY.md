# Security Policy

If you discover a security issue in Hayabusa, do not open a public issue with exploit details.

Please report it privately to the maintainers first so the issue can be reviewed and fixed responsibly.

When reporting:

- describe the affected component
- include reproduction steps if you have them
- include impact and any suggested mitigation

If no private contact channel is listed yet, open a minimal public issue that only asks for a private security contact and does not include sensitive details.
